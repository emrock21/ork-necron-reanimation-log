# Ork vs Necron Reanimation Log

This contract records Ork encounters with Necron forces, focusing on  
their unique technologies:

- Reanimation Protocols  
- Quantum Phase Shifts  
- Dynasty Markers  
- Living Metal behavior  

These elements are exclusive to Necrons and cannot be applied to any  
other faction.
